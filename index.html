Who made my code slower?

We write tests so we can have freedom from fear. When a commit breaks an old test, it tells us where to look to fix the issue. But when we test the performance of our project, how can we know when and where the changes in performance occurred?

What is vbench?

vbench is a super-lightweight Python library to catch performance regressions. It integrates with git to run performance benchmarks for every revision of your source repository, persisting the results in SQLite and generating graphs with matplotlib.

vbench started as a way to track performance in pandas.
